Projeto feito para estudo, para praticar o que foi aprendido no curso Full Stack Club. Nesse projeto busquei praticar o CRUD e Thymeleaf.

## Tecnologias Utilizadas
- Java 21
- Maven
- Spring Boot
- Thymeleaf

## Para rodar o projeto
- Ferramenta: Java v21, Thymeleaf, Spring Boot v3.5.9.
- Gerenciador de pacotes: Maven v4 .
Passo a passo
1. ***Clone o repositório***: ```bash git clone https://github.com/jose-wolf/agenda-contatos.git```
2. cd agenda-contatos
3. mvn clean install
4. mvn spring-boot:run
